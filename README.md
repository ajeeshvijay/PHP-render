# PHP-render
Prints human-readable information about a PHP variable.

[https://ajeeshvijay.github.io/PHP-render/](https://ajeeshvijay.github.io/PHP-render/)

### Include the file

```javascript
<?php
	include('./FILE-ROOT/PHP-render.php');
    
	$testArray = array('zero', 'one', 'two', 'three');
	render($testArray);
?>
```
